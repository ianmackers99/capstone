# Capstone queries and results.xlsx
This Excel spreadsheet contains one tab for each of the eight functions

Each tab contains the initial dataset and the queries (**predictions**) which I submitted for each function over the course of the project

The **Datapoints** are numbered beginning at 1 for the first datapoint in the initial dateset supplied by the faculty through to the last datapoint submitted in week 13 of the project.

The **Preds** column represents the input submitted for processing regardless of source

The **Result** column represents the output generated by the black-box function for those input parameters

The **Improvement Over Time** plot shows exactly that.

The **Excluded from plot** column contains those datapoints which are so far out of range they render the Improvement Over Time plot unusable and so are excluded from it

The **Best Result** and **Best Strategy** are highlighted in red

The **Highest Datapoint** column shows the points at which the result improved for that function

## LLMs prompts

I used LLMs to recommend sample points or hyperparameters for some functions during weeks 8, 9, 10, 11 and 12

The **LLM prompts** subdirectory provides more detail on those prompts and the responses

Note that I didn't always use the LLM recommendations. I describe the cases where I did use them in the **Week by week strategy** section of the main repo [Readme](https://github.com/ianmackers99/capstone/tree/main) file
